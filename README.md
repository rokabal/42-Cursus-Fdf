## Isometric projection
u = x*cos(α) + y*cos(α+120°) + z*cos(α-120°)
v = x*sin(α) + y*sin(α+120°) + z*sin(α-120°)
For α = 30°:
u = x*cos(30°) + y*cos(150°)
v = x*sin(30°) + y*sin(150°) - z
